**복돌은 안 됩니다.**

1. [CurseForge](https://www.curseforge.com/download/app)를 설치하고 실행한다. 
2. 로그인하면 마인크래프트가 있습니다.
3. 마인크래프트를 클릭하면 왼쪽 위에 Create / Import 있을텐데 import 누르고
4. 오른쪽 Use Profile Code에 2BzkfmVC 입력. 코드는 유효기간 (7일)이 있으니 안되면 쫑귿을 부를 것.
5. 다운로드.
6. 실행하고 멀티플레이 -> 서버추가
7. 211.216.117.123 입력

+엘리스에게 갠톡으로 화이트리스트 요청하
