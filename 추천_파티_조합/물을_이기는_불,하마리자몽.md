## ◆ 개요
모두가 한 번쯤 써보고 싶은 리자몽.  
오늘은 메가리자몽Y가 주축인 파티입니다.  

| 유형      | 사용 기믹   | 난이도 |
|-----------|----------|-----------|
| 전개     | 메가진화     | ◆◆◇◇◇ | 

전개 파티지만 대면 성향도 강합니다.  
메가리자몽Y의 깡파워로 어떻게든 됩니다.  
리자몽에게 치명적인 기술이 나올 거 같을 때 하마돈 꺼낼 줄만 알면 됩니다.

<br>

## ◆ 엔트리
### 메가리자몽Y
![와자몽](https://static.wikia.nocookie.net/pokemon/images/9/9f/0006M2.png/revision/latest?cb=20241004213150)  
**메가리자몽Y** @리자몽나이트Y  
조심 / V_VVVV / 맹화->가뭄  
H252 B116 C100 D4 S36  
화염방사 / 솔라빔 / 에어슬래시 / 날개쉬기  

메이저 포켓몬들의 암석봉인에 1타가 안 날 만큼 내구 보정된 메가리자몽입니다.  
가뭄 특성으로 **쾌청을 깔고 솔라빔을 쏘면 역으로 물 타입까지 녹일 수 있습니다.**  
무식한 파워를 믿고 선봉으로 내는 경우가 많습니다.

### 하마돈
![하마돈](https://static.wikia.nocookie.net/pokemon/images/2/2b/0450.png/revision/latest?cb=20241006044922)  
**하마돈** @울퉁불퉁멧  
장난꾸러기 / 6V / 모래날림  
H252 B252 D4  
지진 / 스텔스록 / 하품 / 날려버리기  

이 파티에서 하마돈의 역할은 선봉이 아닌 리자몽의 보조입니다.  
리자몽의 세 약점 중 쾌청으로 커버 가능한 물을 뺀 나머지 **바위, 전기 타입을 받아 줍니다.**  
활용법은 늘 비슷합니다. 하품으로 귀찮게 하며 대면 조작하기.  
메가리자몽에게 곤란한 상대가 보이면 선봉으로 냅니다.

### 따라큐
![따라큐](https://static.wikia.nocookie.net/pokemon/images/9/97/0778.png/revision/latest?cb=20241009010047)  
**따라큐** @생명의구슬  
고집or명랑 / 6V / 탈  
H4 A252 S252  
섀도크루 / 야습 / 치근거리기 / 칼춤

어디 넣어도 강력한 따라큐입니다만 이번에는 단순히 강해서 들어간 것만은 아닙니다.  
페어리 타입의 천적인 **강철 타입만 리자몽이 녹여주면 따라큐는 막히는 것 없이 스윕이 가능하죠.**  
잠재-얼음을 뺏긴 리자몽이 타점이 안 나와 곤란해하는 드래곤 타입을 잡아주기도 합니다.

<br>

## ◆ 운용법
쉽습니다. **선봉은 일단 리자몽을 냅니다.**  
만약 상대가 바위, 전기 타입 기술을 쓸 것 같다면 하마돈으로 교체합니다.  
리자몽이 강철 타입만 치우면 나머지는 따라큐가 정리할 수 있습니다.  
**따라큐가 강철 타입 포켓몬을 맞닥뜨리는 상황만은 피합시다.**
