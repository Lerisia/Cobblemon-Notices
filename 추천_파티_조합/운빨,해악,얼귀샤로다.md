## ◆ 개요
먼 옛날부터 포켓몬 실전을 하는 사람들은 게임이 안 풀리면 악마의 손을 잡곤 했습니다.  
**버티기만 하면 랜덤한 능력치가 2랭크 올라가고 랜덤한 능력치가 1랭크 떨어지는 얼음귀신.** 애석하게도 대타출동과 방어도 배웁니다.  
이 얼음귀신의 손을 잡고 운빨로 도저히 못 올리겠는 레이팅 점수대 구간을 어거지로 미는 거죠.  
**운빨로 이길 수 있을 뿐만 아니라, 상대를 매우 불쾌하게 만들기까지 하는 일석이조의 파티. '해악파티' 얼귀샤로다**를 만나보겠습니다.

난이도랄 게 없습니다. 운이 좋으면 이깁니다.

## ◆ 엔트리
### 얼음귀신
![얼음귀신](https://static.wikia.nocookie.net/pokemon/images/f/f1/0362.png/revision/latest?cb=20241006025629)  
**얼음귀신** @먹다남은음식  
겁쟁이 / VZVVVV / 변덕쟁이  
H172 B84 S252  
얼음숨결or프리즈드라이or냉동빔 / 절대영도 / 대타출동 / 방어

얼귀샤로다 해악파티의 알파이자 오메가. 얼음귀신입니다.  
샤로다 등으로 마비를 걸어 기점을 잡고 나가거나, 얼음 기술을 받으며 나갑니다.  
그리고 **대타출동, 방어, 먹밥을 이용해 계속 버티면서 좋은 능력치가 올라가길 기도합니다.**  
적당히 버티다보면 괴물이 된 얼음귀신으로 스윕합니다.


### 샤로다
![샤로다](https://static.wikia.nocookie.net/pokemon/images/7/75/0497.png/revision/latest?cb=20241007011536)  
**샤로다** @기합의띠  
겁쟁이 / VZVVVV / 심술꾸러기  
H252 C4 S252   
리프스톰 / 씨뿌리기 / 뱀눈초리 / 도발 // 대타출동, 미러코트 등

이 파티의 숨겨진 주인공입니다.  
**명중 100의 마비 기술 뱀눈초리와, 후속 포켓몬에게도 흡혈을 주는 씨뿌리기**로 얼음귀신이 나오기 최적의 환경을 조성합니다.  
뱀눈초리와 씨뿌리기를 썼으면 빠르게 퇴장하고 얼음귀신으로 교대하는 게 좋습니다.  

### 한카리아스
![한카리아스](https://static.wikia.nocookie.net/pokemon/images/5/5a/0445.png/revision/latest?cb=20241006044159)  
**한카리아스** @전기구슬  
명랑 / 6V / 까칠한피부  
H172 D84 S252  
지진 / 암석봉인 / 내던지기 / 스텔스록 // 모래뿌리기

자기 자신은 마비에 안 걸리는 점을 이용해 전기구슬을 들고, **내던지기로 마비를 거는 기점마련형 한카리아스**입니다.  
스텔스록도 그렇고 생각보다 거지같은 기술을 많이 가지고 있습니다.  
**얼음 기술을 유도**해서 좋은 얼음귀신 후내밀기 타이밍을 만들어주는 건 덤입니다.  

<br>

## ◆ 운용법
우선 상대 파티 보고 샤로다나 한카리아스를 선봉으로 냅니다. 스텔스록, 마비, 씨뿌리기 등을 걸며 거지같이 굴어줍니다.  
그러다가 적당한 타이밍에 샤로다를 퇴장시키거나, 한카리아스에게 들어오는 얼음 기술을 받으며 얼음귀신이 나갑니다.  
또 다시 대타출동과 방어를 번갈아 써가면서 거지같이 굴다가 랭크업이 잘 되면 상대를 쓸어버립니다.  

<br>

## ◆ 번외
레귤레이션 상 현재는 채용이 불가능하지만, 이 파티의 세 번째 멤버로 가장 유명한 건 원래 크레세리아입니다. 그 다음은 볼트로스구요.  
나중을 위해 두 포켓몬의 샘플을 소개하며 마무리합니다.  

![크레세리아](https://static.wikia.nocookie.net/pokemon/images/3/37/0488.png/revision/latest?cb=20241006053020)  
**크레세리아** @울퉁불퉁멧  
대담 / VZVVVV / 부유  
H252 B252 D4   
얼다바람 / 전기자석파 / 달의불빛 / 초승달춤

높은 내구와 달의불빛으로 버티며 얼다바람으로 스피드를 깎고, 전기자석파로 마비를 겁니다.  
초승달춤으로 자신은 퇴장하면서 체력이 떨어진 얼음귀신을 되살릴 수도 있는, 얼귀샤로다의 완벽한 서포터입니다.  

![볼트로스](https://static.wikia.nocookie.net/pokemon/images/c/c6/0642.png/revision/latest?cb=20241007034556)  
**볼트로스** @생명의구슬  
겁쟁이 / VZVVVV / 짓궂은마음  
H4 C252 S252   
10만볼트 / 사이코키네시스 / 기합구슬 / 전기자석파  

111이라는 높은 스피드 종족값, 강력한 화력, 그리고 짓맘 전기자석파라는 훌륭한 스펙을 가지고 있습니다.  
전기자석파를 걸면서 짜증나게 하는 것도 잘 하지만, 여차하면 그냥 화력으로 밀어버리는 것도 잘 합니다.
