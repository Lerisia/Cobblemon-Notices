메짱이의 파트너로 수비적인 너트령 대신 공격적인 종이신도를 써서 사이클 느낌을 아예 뺐습니다.  
전개와 대면이 섞인 느낌의 파티네요. 가볍게 사용해주시면 될 것 같습니다.

## ◆ 엔트리
### 메가번치코
![메가번치코](https://projectpokemon.org/images/normal-sprite/blaziken-mega.gif)  
**메가번치코** @번치코나이트  
고집 / 6V / 가속->가속  
A252 B4 S252  
플레어드라이브 / 인파이트 / 번개펀치 / 칼춤  

보만다와 랜드로스가 없는 탓에 만능이 된 메가번치코입니다.  
번개펀치로 물타입까지 잡아내면서 사이클 파티를 붕괴시키는 역할을 수행합니다.  

### 메가대짱이
![메가대짱이](https://projectpokemon.org/images/normal-sprite/swampert-mega.gif)  
**메가대짱이** @대짱이나이트  
고집 / 6V / 습기->쓱쓱  
A252 D4 S252  
폭포오르기 / 지진 / 냉동펀치 / 대타출동  

비팟 에이스 메가대짱이.  
내구, 스피드 모두 훌륭하지만 늘 에이스 치고는 딜이 2% 부족한 느낌이에요.  
이 부족함을 빠른 스피드 기반의 대타출동으로 채웁니다.  
대타출동은 변화기를 무효화하거나 기습 심리전을 이기거나 교체 예측해서 깔거나 쓸모가 매우 많습니다.  

### 패리퍼
![패리퍼](https://projectpokemon.org/images/normal-sprite/pelipper.gif)  
**패리퍼** @기합의띠  
조심 / 6V / 잔비  
B4 C252 S252  
파도타기 / 폭풍 / 유턴 / 순풍or냉동빔

비팟의 진짜 에이스 패리퍼입니다.  
스스로 비 깔고 명중 100으로 던지는 자속 폭풍은 위력도 일관성도 우수합니다.  
기합의띠라 불리대면이면 유턴으로 빠지고 나중에 한 번 더 비를 깔 수도 있구요.  
순풍은 사고로 비 턴이 끌렸을 때 비 대신 깔고 퇴장할 수도 있습니다. 종이신도 서포팅엔 이쪽이 더 좋기도 하죠.

## ◆ 선출
![메짱이](https://static.wikia.nocookie.net/pokemon/images/c/c7/260m%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20200213112702&path-prefix=ko) ![패리퍼](https://static.wikia.nocookie.net/pokemon/images/3/35/279%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20200213105624&path-prefix=ko)  
기본 선출. 패리퍼가 비를 내리고 메가대짱이가 스윕합니다.  
나머지 한 자리는 ![따라큐](https://static.wikia.nocookie.net/pokemon/images/b/b4/778%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20161125134544&path-prefix=ko)나 ![종이신도](https://static.wikia.nocookie.net/pokemon/images/b/be/798%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20161125134615&path-prefix=ko)를 데려갑니다.  
기본 선출이지만 종이신도, 버섯모, 뽀록나 등 절대 못 잡아내는 포켓몬이 일부 있습니다. 이런 포켓몬이 보이면 선출을 피합시다.  

![카푸나비나](https://static.wikia.nocookie.net/pokemon/images/d/de/786%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20161125134556&path-prefix=ko) ![메가번치코](https://static.wikia.nocookie.net/pokemon/images/8/86/257m%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20200213112701&path-prefix=ko)  
비 파티가 막힐 때 데려가는 불꽃프레임 선출입니다.  
나머지 한 자리는 역시 ![따라큐](https://static.wikia.nocookie.net/pokemon/images/b/b4/778%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20161125134544&path-prefix=ko)나 ![종이신도](https://static.wikia.nocookie.net/pokemon/images/b/be/798%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20161125134615&path-prefix=ko)를 데려갑니다.  

![메짱이](https://static.wikia.nocookie.net/pokemon/images/c/c7/260m%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20200213112702&path-prefix=ko) ![카푸나비나](https://static.wikia.nocookie.net/pokemon/images/d/de/786%EC%83%88%EB%B0%95%EC%8A%A4%EC%95%84%EC%9D%B4%EC%BD%98.png/revision/latest?cb=20161125134556&path-prefix=ko)  
자주 나오는 선출은 아닙니다만 비 파티를 꺼낼 수 없는 상황에 상대 파티에 번치코가 있으면 선출합니다.
